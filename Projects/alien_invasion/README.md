# Alien Invasion Game

A 2D arcade-style game built with **Pygame**, based on the textbook *Python Crash Course* by Eric Matthes.  
The player controls a ship to shoot down alien fleets, practicing object-oriented programming and event-driven logic.

## Features
- Player movement and bullet firing
- Alien fleet generation and movement
- Collision detection between bullets and aliens
- Score tracking and game progression

## Tech Stack
- Python
- Pygame

Run the game with:
```bash
python alien_invasion.py
