# Data Visualization Activities

A collection of Python scripts and notebooks exploring data visualization techniques.  
Demonstrates use of **Matplotlib**, **Plotly**, and **Pygal** for charts and interactive graphs.

## Activities
- Line and scatter plots
- Histograms
- Random walk simulations
- Dice roll probability visualizations

## Tech Stack
- Python
- Matplotlib
- Plotly
- Pygal

Run examples with:
```bash
python cubes.py
